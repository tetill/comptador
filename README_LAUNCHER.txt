APLICATIU LA REINAIXENÇA - OPTIMIZED VERSION
=====================================================

INSTRUCCIONS PER EXECUTAR L'APLICACIÓ:

1. DOBLE CLIC FÀCIL:
   - Fes doble clic a "APLICATIU_RENAIXENCA_15_OPTIMIZED.bat"
   - Això obrirà l'aplicació amb una finestra de comandos visible

2. EXECUCIÓ SILENCIOSA (sense finestra de comandos):
   - Fes doble clic a "Launch_APLICATIU_RENAIXENCA.vbs"
   - Això obrirà l'aplicació sense mostrar la finestra de comandos

3. EXECUCIÓ MANUAL:
   - Obre una terminal/PowerShell
   - Navega a n:/
   - Executa: python APLICATIU_RENAIXENCA_15_OPTIMIZED.py

MILLORES IMPLEMENTADES:
======================
✓ Optimització de rendiment amb caching
✓ Millor gestió de memòria
✓ Tipus de dades definits (type hints)
✓ Estructura de codi més clara i modular
✓ Millor gestió d'errors
✓ Documentació completa
✓ Compatibilitat amb VLC i Pygame
✓ Drag & drop per fitxers d'àudio

FITXERS INCLOSOS:
================
- APLICATIU_RENAIXENCA_15_OPTIMIZED.py (codi principal optimitzat)
- APLICATIU_RENAIXENCA_15_OPTIMIZED.bat (launcher amb finestra)
- Launch_APLICATIU_RENAIXENCA.vbs (launcher silenciós)
- test_optimized.py (tests de funcionalitat)
- README_LAUNCHER.txt (aquest fitxer)

REQUISITS:
==========
- Python 3.7+
- tkinter (inclòs amb Python)
- pygame (opcional, per àudio)
- python-vlc (opcional, per àudio avançat)
- tkinterdnd2 (opcional, per drag & drop)

L'aplicació funcionarà amb les funcionalitats bàsiques encara que no tinguis
tots els mòduls opcionals instal·lats.
